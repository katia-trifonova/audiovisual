Web Audio API

Web Audio API

  1. AudioContext
  2. MediaElementAudioSourceNode
  3. AnalyserNode

AudioContext

  • Граф, состоящий из связанных узлов AudioNode
  • Всё, что происходит при обработке звука, происходит внутри AudioContext
  • Может выступать как слушатель событий и реализует интерфейс EventTarget.

AudioContext

const audioCtx = new AudioContext();

		
Кроссбраузерный вариант:


		const AudioContext = window.AudioContext || 
							 window.webkitAudioContext;
		const audioCtx = new AudioContext();

AudioContext.destination

  • Ссылка на AudioDestinationNode — точку назначения для всего аудио в этом контексте.

MediaElementAudioSourceNode

  • AudioNode, созданный из <audio>, <video> или new Audio().
  • Не имеет входов, только один выход.
  • Может быть создан методом AudioContext.createMediaElementSource.

MediaElementAudioSourceNode

const audioEl = new Audio();
		audioEl.src = "kek.mp3"; 
		const audioSrc = audioCtx.createMediaElementSource(audioEl);

AnalyserNode

  • Это AudioNode, который на входе принимает аудиопоток и передаёт его без изменений на выход.
  • При этом генерирует данные о частоте.
  • Их можно использовать для создания визуализаций

AnalyserNode

AnalyserNode

const analyser = audioCtx.createAnalyser();
		audioEl.src = "kek.mp3"; 
		const audioSrc = audioCtx.createMediaElementSource(audioEl);

Подключаем аудио

audioSrc.connect(analyser);
		audioSrc.connect(audioCtx.destination);
